Our Model

The Shoplifting detection model uses advanced **YOLO (You Only Look Once)** image recognition technology to analyze real-time video feeds and identify suspicious behavior in retail environments. YOLO allows us to detect multiple objects, such as individuals or items, in a single frame, making it ideal for tracking potential theft in stores. The confusion matrix shown in the image provides a summary of the model's performance.

The confusion matrix includes four key metrics:

  • True Positives (TP): The number of correctly identified shoplifting incidents, which is 103.
  • True Negatives (TN): The number of correctly identified non-shoplifting cases, which is 231.
  • False Positives (FP): The number of non-shoplifting cases incorrectly identified as shoplifting, which is 129.
  • False Negatives (FN): The number of shoplifting incidents incorrectly identified as non-shoplifting, which is 137.

The matrix highlights the model's accuracy in distinguishing between shoplifting and non-shoplifting behavior. In this case, the model correctly identified 231 non-shoplifting incidents and 103 shoplifting incidents, but it also made errors, misclassifying 129 non-shoplifting cases as shoplifting and 137 shoplifting incidents as non-shoplifting. This confusion matrix is a useful tool for evaluating the effectiveness of the model and identifying areas for improvement. The color intensity in the matrix visually represents the frequency of each classification, with darker shades indicating higher counts.
